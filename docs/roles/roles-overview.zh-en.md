# Role-based Reading Guide (中文 + English)

导航 / Navigation: [返回项目首页](../../README.md) | [文档首页](../README.md)

## 目的 / Purpose

中文：
这份指南按团队角色提供最短阅读路径，帮助新成员在最短时间内完成上手并进入有效协作。

English:
This guide provides shortest reading paths by role so new team members can onboard fast and collaborate effectively.

## 角色入口 / Role Entry

1. 开发者 / Developer
- 阅读: [developer-playbook.zh-en.md](developer-playbook.zh-en.md)
- 关注: API、Runtime、Worker、数据模型、前端联调

2. 测试工程师 / QA
- 阅读: [qa-playbook.zh-en.md](qa-playbook.zh-en.md)
- 关注: E2E 场景、回归点、故障注入、验收清单

3. 运维工程师 / Ops
- 阅读: [ops-playbook.zh-en.md](ops-playbook.zh-en.md)
- 关注: 部署、监控、告警、发布、回滚

4. 产品经理 / PM
- 阅读: [pm-playbook.zh-en.md](pm-playbook.zh-en.md)
- 关注: 功能边界、可观测指标、里程碑与验收

## 通用基础文档 / Common Baseline

所有角色建议至少阅读：
1. [quick-start.zh-en.md](../guides/quick-start.zh-en.md)
2. [frontend-guide.zh-en.md](../guides/frontend-guide.zh-en.md)
3. [runtime-run-worker.zh-en.md](../guides/runtime-run-worker.zh-en.md)
4. [testing-playbook.zh-en.md](../guides/testing-playbook.zh-en.md)
